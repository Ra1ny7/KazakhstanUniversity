KazakhstanUniversity — Сравнение вузов Казахстана

Кратко: это веб-приложение для поиска, сравнения и просмотра информации о вузах Казахстана. Backend — на Flask + Jinja (Python), фронтенд — HTML + JavaScript, шаблоны — Jinja. Проект предоставляет список вузов, страницу университета с детальной информацией, возможность добавить в «Избранное» и перенести выбранный университет в таблицу сравнения.

Примечание: вы написали «джава» — в README я использую JavaScript для фронтенда. Если вы имели в виду Java (JVM), напишите — я быстро поправлю инструкции и разделы сборки.
